Aplikasi streamlit untuk mengekstrak informasi dari file audio yang diupload
anda bisa memasukan prompt untuk melakukan apa yang ingin anda lakukan

pastikan anda memiliki api key dari https://aistudio.google.com/app/prompts/new_chat

menggunakan model gemini 2.0 flash dengan token kurang lebih 1 juta

file yang diupload berupa mp4 dengan durasi maskimal 20 menit

pastikan anda telah menginstall python sebelumnyas
